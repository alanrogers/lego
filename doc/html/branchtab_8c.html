Chi-squared difference between observed and expected.

The Chi-squared statistic is (obs - expected)^2/variance. To estimate the variance, note that the expected number of mutations is u*nnuc*Poisson(B), where B is a random variable, the branch length per nucleotide site contributing to a given site pattern. This is u*nnuc times a mixture of Poisson distributions. It's mean is u*nnuc*Mean(B), and its variance is u*nnuc*Mean(B) + nnuc*u*u*Var(B).
