Detailed Description

Map parameter addresses to parameter structures.

This class uses a left-leaning binary tree to build a map that associates parameter addresses with structures that include a pointer to the current value, the lower and upper bounds, and the type of the parameter. The interface allows parameters to be inserted and later looked up by address.
