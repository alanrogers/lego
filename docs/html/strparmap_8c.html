Detailed Description

Map parameter names to parameter structures.

This class uses a left-leaning binary tree to build a map that associates parameter names with structures that include a pointer to the current value, the lower and upper bounds, and the type of the parameter. The interface allows parameters to be inserted and later looked up by name.
